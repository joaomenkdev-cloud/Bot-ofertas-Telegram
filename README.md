# Bot-ofertas-Telegram
Bot automatizado para Telegram que faz web scraping de ofertas, filtra produtos de tecnologia e gaming, e envia automaticamente para grupos usando sistema de fila inteligente. Evita spam enviando 1 oferta a cada 10 minutos. Nunca duplica ofertas (hash SHA256). Busca contínua 24/7 com MySQL para persistência.
